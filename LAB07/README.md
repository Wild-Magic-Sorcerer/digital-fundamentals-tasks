1. Загрузите пакет свободных игр в Python (https://github.com/grantjenks/free-python-games).
2. Установите его.
3. Запустите какую-нибудь игру на ваш выбор (попробуйте запустить пакет напрямую через командную строку, не вызывая модуль игр через python).
4. Загрузите пакет игры в лабиринт (https://github.com/josokw/Maze).
5. Установите его (поменяйте минимальную версию cmake в CMakeLists.txt с 2.8 на, например, 3.8).
6. Поиграйте в него (можно создать свой лабиринт).
